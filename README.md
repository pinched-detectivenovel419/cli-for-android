## 📦 What This Is

cli-for-android is a simple command-line tool for Android work. It helps you handle common Android tasks from one place. You can use it to work with devices, apps, files, and build tools without jumping between many tools.

This README is written for Windows users who want to download the app and start using it fast.

## 🖥️ What You Need

Before you install, check these basics:

- Windows 10 or Windows 11
- A working internet connection
- At least 200 MB of free space
- Permission to run apps on your PC
- Android USB drivers if you plan to connect a phone or tablet

If you want to use device features, your Android device should have USB debugging turned on.

## 📱 First-Time Setup

When you open cli-for-android for the first time, follow these steps:

1. Keep the app in a folder you can find later
2. If the app asks for a device path or tools path, use the default option first
3. Connect your Android device with a USB cable if you want to work with a device
4. Turn on USB debugging on your phone
5. Allow the computer prompt on your phone
6. Open the app again if it did not detect the device right away

If you use Android Studio or platform tools already, the app may ask where those tools are stored. Point it to the folder that holds `adb` and other Android tools.

## 🔌 Connect an Android Device

To use device features:

1. Open Settings on your Android device
2. Go to About phone
3. Tap Build number seven times if you need to turn on Developer options
4. Open Developer options
5. Turn on USB debugging
6. Connect the phone to your PC with a USB cable
7. Allow the computer on the phone when asked

After that, open cli-for-android and check whether it sees the device.

## 🧩 Troubleshooting

If the app does not start:

- Try running it as an administrator
- Check that the file finished downloading
- Make sure you extracted the ZIP file
- Check whether your Windows antivirus blocked the file

If your device does not show up:

- Use a data USB cable, not a charge-only cable
- Turn on USB debugging
- Unlock your phone
- Accept the computer prompt on the device
- Try another USB port

If the app cannot find Android tools:

- Make sure platform tools are installed
- Check the path you entered
- Point the app to the folder that contains `adb.exe`

If a command fails:

- Disconnect and reconnect the phone
- Close other Android tools
- Open the app again
- Check that the device is awake and unlocked
